# Projeto To-do List com Docker

## Como rodar o projeto

1. Clone o repositório:
```bash
git clone <url-do-seu-repositorio>
cd todo-app
```

2. Execute o Docker:
```bash
docker-compose up --build
```

3. Acesse:
- Frontend: http://localhost:5000
- Backend: http://localhost:5000

## Endpoints da API
- GET /tasks
- POST /tasks
- DELETE /tasks/:id
